EEGReading

public struct EEGReading : Sendable

EEG(뇌전도) 센서 읽기값을 나타내는 구조체입니다.

이 구조체는 2채널 EEG 데이터와 관련 메타데이터를 포함합니다. 전압 값은 마이크로볼트(µV) 단위로 표현되며, 원시 ADC 값도 함께 제공됩니다.

예시

let eegReading = EEGReading(
    channel1: 15.3,
    channel2: -8.7,
    ch1Raw: 125043,
    ch2Raw: -67834,
    leadOff: false
)
