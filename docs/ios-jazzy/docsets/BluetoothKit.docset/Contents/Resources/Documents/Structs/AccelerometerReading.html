AccelerometerReading

public struct AccelerometerReading : Sendable

3축 가속도계 센서 읽기값을 나타내는 구조체입니다.

이 구조체는 디바이스의 움직임과 방향을 감지하기 위한 X, Y, Z축의 가속도 데이터를 포함합니다.

예시

let accelReading = AccelerometerReading(
    x: 1024,   //
    y: 0,      //
    z: 0       //
)
