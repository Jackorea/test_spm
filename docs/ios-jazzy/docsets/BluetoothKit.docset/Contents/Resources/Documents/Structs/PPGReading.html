PPGReading

public struct PPGReading : Sendable

PPG(광전 용적 맥파) 센서 읽기값을 나타내는 구조체입니다.

PPG는 심박수와 혈류량 모니터링에 사용되는 광학 센서입니다. 적색(Red)과 적외선(IR) LED를 사용하여 혈액의 산소 포화도와 심박수를 측정할 수 있습니다.

예시

let ppgReading = PPGReading(
    red: 125043,
    ir: 134567
)
